Learn Jam 2016


Videos are now live! Check out the YouTube playlist!

Learn Jam 2016 is your chance to test the waters of Game Development, with SGDC's helping hand. We'll be going through multiple engines and exploring the basics of creating games in each of them, and experimenting with a few tools that aid in the process. If you are interested in attending one of the sessions and plan on following along, we do request that you download and install the programs of interest to your computer before the event -- these are some big files.

Learn Jam will be taking place in Peirce 116 on Sunday, the 11th of September.

Lesson schedule is as follows. If you are interested in an event, please follow the links provided and install the latest version to your computer.

  • Unreal Engine 4: 11:30 - 12:45
    • requires account creation
  • Git: 12:45 - 1:30
    • Default settings on installation should be fine
    • If you're not sure download the installer and we'll install it at the event
  • Blender: 2:00 - 3:00
  • Unity 5: 3:15 - 4:45
  • Game Maker: Studio: 5:00 - 6:30
    • requires account creation
    • get "Studio FREE"

Pizza will be provided. We hope you enjoy!
